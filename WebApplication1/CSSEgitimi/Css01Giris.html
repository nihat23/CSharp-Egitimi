CSS Sözdizimi

CSS kural kümesi bir seçici ve bildirim bloğundan oluşur:

Örneğin h1 {color:blue;} şeklinde bir bildirimde bulunursak burada h1 seçici yani css i uygulamak istediğimiz elemandır, küme parantezi içindeki color property yani özelliktir, blue ise bu özelliğe atanan değer (value) dir.

Satır içi (İnline) CSS

Satır içi (İnline) CSS : Bu elementte kullandığımız style="background-color: brown; color: bisque;" kodları ile yazım şekline inline css denir. Bu şekilde de html elementlerine css yazabiliriz.

Html elementlerinde renk ismi yerine renk kodlarını da kullanabiliriz

HTML öğesi için birden fazla stil belirtildiğinde hangi stil kullanılır?

1- Satır içi stili (İnline CSS) (html öğesi içinde h3 style="color: blue;" şeklinde yazarsak) 2- Dış ve iç stil sayfaları (baş bölümünde belirttiğim gibi css lerin sırasına göre) 3- Tarayıcı varsayılan (her tarayıcıcnın html elemanlarına uyguladığı varsayılan css özellikleri vardır)
